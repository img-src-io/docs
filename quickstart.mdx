# Quick Start

This guide will help you upload your first image and serve it with transformations.

## Prerequisites

- An img-src account ([Sign up free](https://img-src.io/sign-up))

---

## Web Dashboard

The fastest way to get started — no code or API key required.

### Step 1: Upload an Image

1. Go to [img-src.io/publish](https://img-src.io/publish) and sign in
2. Drag & drop images or folders into the upload area, or click **Select Files** / **Select Folder**
   - Max **5MB** per file, up to **20 files** at a time
4. Once the upload completes, you'll see **"Publish complete"** with a preview of each image

### Step 2: Get Your CDN URLs

After publishing, each image shows **Available Formats** with ready-to-use CDN URLs:

| Format | URL |
|--------|-----|
| **WEBP** | `https://img-src.io/i/username/photo.webp` |
| **JPEG** | `https://img-src.io/i/username/photo.jpg` |
| **PNG** | `https://img-src.io/i/username/photo.png` |
| **AVIF** | `https://img-src.io/i/username/photo.avif` |
| **JXL** ⚠️ | `https://img-src.io/i/username/photo.jxl` |

Click the copy icon next to any URL to copy it to your clipboard.

### Step 3: Serve with Transformations

Add query parameters to your CDN URL to resize, crop, or adjust quality on the fly:

```
https://img-src.io/i/username/photo.webp?w=800&h=600&fit=cover&q=85
```

<CodeGroup>

```html Original
<img src="https://img-src.io/i/username/photo.jpg" />
```

```html Resized (800px width)
<img src="https://img-src.io/i/username/photo.jpg?w=800" />
```

```html Thumbnail (200x200, cover)
<img src="https://img-src.io/i/username/photo.jpg?w=200&h=200&fit=cover" />
```

```html WebP with quality
<img src="https://img-src.io/i/username/photo.webp?w=800&q=85" />
```

</CodeGroup>

<Tip>
  Output format is determined by the file extension in the URL. Change `.jpg` to `.webp` or `.avif` to convert formats.
</Tip>

<Warning>
  JXL format has very limited browser support. See the [image transformation guide](/guides/image-transformation#output-format) for details and fallback strategies.
</Warning>

## API & SDK

For automated workflows, use the API directly or through an SDK.

### Step 1: Get Your API Key

1. Go to [img-src.io/settings](https://img-src.io/settings)
2. Click **Create API Key**
3. Give it a name and copy the key (it won't be shown again)

<Warning>
  Keep your API key secure. Don't commit it to version control or expose it in client-side code.
</Warning>

### Step 2: Upload an Image

<CodeGroup>

```bash cURL
curl -X POST https://api.img-src.io/api/v1/images \
  -H "Authorization: Bearer imgsrc_YOUR_API_KEY" \
  -F "file=@photo.jpg" \
  -F "path=photos/vacation.jpg"
```

```typescript TypeScript
import { Imgsrc } from "@img-src/sdk";
import fs from "fs";

const client = new Imgsrc({ bearerAuth: "imgsrc_YOUR_API_KEY" });

const { uploadResponse } = await client.images.uploadImage({
  file: {
    fileName: "photo.jpg",
    content: fs.readFileSync("photo.jpg"),
  },
  targetPath: "photos/vacation.jpg",
});

console.log(uploadResponse?.url);
// https://img-src.io/i/username/photos/vacation.jpg
```

```python Python
import os
from img_src import Imgsrc, models

client = Imgsrc(bearer_auth="imgsrc_YOUR_API_KEY")

with open("photo.jpg", "rb") as f:
    res = client.images.upload_image(request=models.UploadImageRequestBody(
        file=models.File(file_name="photo.jpg", content=f.read()),
        target_path="photos/vacation.jpg",
    ))

print(res.upload_response.url)
# https://img-src.io/i/username/photos/vacation.jpg
```

```go Go
import (
    "context"
    "fmt"
    "io"
    "os"

    sdkgo "github.com/img-src-io/sdk-go"
    "github.com/img-src-io/sdk-go/models/operations"
)

s := sdkgo.New(sdkgo.WithSecurity("imgsrc_YOUR_API_KEY"))

file, err := os.Open("photo.jpg")
if err != nil {
    log.Fatal(err)
}
defer file.Close()

content, _ := io.ReadAll(file)
res, err := s.Images.Upload(context.Background(), &operations.UploadImageRequestBody{
    File: &operations.File{
        FileName: "photo.jpg",
        Content:  content,
    },
    TargetPath: sdkgo.String("photos/vacation.jpg"),
})
if err != nil {
    log.Fatal(err)
}

fmt.Println(res.UploadResponse.URL)
// https://img-src.io/i/username/photos/vacation.jpg
```

```rust Rust
use img_src::apis::{configuration::Configuration, images_api};
use std::path::PathBuf;

let mut config = Configuration::new();
config.bearer_access_token = Some("imgsrc_YOUR_API_KEY".to_string());

let res = images_api::upload_image(
    &config,
    Some(PathBuf::from("photo.jpg")),
    Some("photos/vacation.jpg"),
    None,
).await?;

println!("{}", res.url);
// https://img-src.io/i/username/photos/vacation.jpg
```

</CodeGroup>

The response includes your CDN URL:

```json
{
  "id": "img_abc123",
  "url": "https://img-src.io/i/username/photos/vacation.jpg",
  "paths": ["photos/vacation.jpg"],
  "content_type": "image/jpeg",
  "size": 1234567,
  "width": 4000,
  "height": 3000,
  "created_at": 1704067200
}
```

### Step 3: List Your Images

<CodeGroup>

```bash cURL
curl https://api.img-src.io/api/v1/images \
  -H "Authorization: Bearer imgsrc_YOUR_API_KEY"
```

```typescript TypeScript
const { imageListResponse } = await client.images.listImages(10);
console.log(imageListResponse?.images);
```

```python Python
res = client.images.list_images(limit=10)
print(res.image_list_response.images)
```

```go Go
res, err := s.Images.List(ctx, sdkgo.Int64(10), nil, nil)
if err != nil {
    log.Fatal(err)
}
fmt.Println(res.ImageListResponse.Images)
```

```rust Rust
let res = images_api::list_images(&config, Some(10), None, None).await?;
println!("{:?}", res.images);
```

</CodeGroup>

### Step 4: Delete an Image

<CodeGroup>

```bash cURL
curl -X DELETE https://api.img-src.io/api/v1/images/img_abc123 \
  -H "Authorization: Bearer imgsrc_YOUR_API_KEY"
```

```typescript TypeScript
await client.images.deleteImage("img_abc123");
```

```python Python
client.images.delete_image(id="img_abc123")
```

```go Go
_, err := s.Images.Delete(ctx, "img_abc123")
if err != nil {
    log.Fatal(err)
}
```

```rust Rust
images_api::delete_image(&config, "img_abc123").await?;
```

</CodeGroup>

## Transformation Parameters

| Parameter | Description | Example |
|-----------|-------------|---------|
| `w` | Width in pixels | `w=800` |
| `h` | Height in pixels | `h=600` |
| `fit` | Fit mode | `fit=cover` |
| `q` | Quality (1-100) | `q=85` |

### Fit Modes

- **cover** - Fill the dimensions, cropping if needed
- **contain** - Fit within dimensions, preserving aspect ratio
- **fill** - Stretch to fill dimensions exactly
- **scale-down** - Like contain, but never upscale
