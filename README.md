# img-src Documentation

Official documentation for [img-src](https://img-src.io) - Serverless image processing and delivery API.

## Development

```bash
# Install Mintlify CLI
npm i -g mintlify

# Start local dev server
mintlify dev
```

Open http://localhost:3000 to preview.

## Deployment

Documentation is automatically deployed to [docs.img-src.io](https://docs.img-src.io) via Mintlify.

## Structure

```
├── mint.json              # Mintlify configuration
├── introduction.mdx       # Service introduction
├── quickstart.mdx         # Quick start guide
├── api-reference/         # API endpoint documentation
├── guides/                # How-to guides
├── sdk/                   # SDK documentation
└── examples/              # Integration examples
```
