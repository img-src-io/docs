---
title: Delete Preset
api: DELETE /api/v1/settings/presets/{name}
description: Delete a transformation preset (Pro only)
---

<Note>
  This endpoint requires a Pro plan subscription.
</Note>

<RequestExample>
```bash cURL
curl -X DELETE https://api.img-src.io/api/v1/settings/presets/thumbnail \
  -H "Authorization: Bearer imgsrc_YOUR_API_KEY"
```
</RequestExample>

<ResponseExample>
```json 200 OK
{
  "success": true,
  "message": "Preset deleted"
}
```
</ResponseExample>

<Warning>
  After deletion, URLs using this preset will return a 400 error. Make sure to update your code before deleting.
</Warning>
