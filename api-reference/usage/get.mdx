---
title: Get Usage
api: GET /api/v1/usage
description: Get usage statistics for the current billing period
---

<RequestExample>
```bash cURL
curl https://api.img-src.io/api/v1/usage \
  -H "Authorization: Bearer imgsrc_YOUR_API_KEY"
```
</RequestExample>

## Usage Metrics

| Metric | Description |
|--------|-------------|
| `transformations` | Number of image transformations this period |
| `storage` | Total storage used across all images |
| `bandwidth` | CDN bandwidth consumed this period |

## Plan Limits

| Metric | Free | Pro |
|--------|------|-----|
| Transformations | 1,000/month | 10,000/month |
| Storage | 10 GB | 50 GB |
| Bandwidth | Unlimited | Unlimited |
