---
title: Search Images
api: GET /api/v1/images/search
description: Search images by path or metadata
---

<RequestExample>
```bash cURL
curl "https://api.img-src.io/api/v1/images/search?q=vacation&limit=20" \
  -H "Authorization: Bearer imgsrc_YOUR_API_KEY"
```
</RequestExample>

<ResponseExample>

</ResponseExample>
